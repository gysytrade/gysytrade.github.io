# Guangyuan Shengyuan International Trade Website

This is the official corporate website for Guangyuan Shengyuan International Trade Co., Ltd.

## Tech Stack
- **Framework**: Astro 5
- **Styling**: Tailwind CSS
- **Design**: Custom Premium Theme

## Project Structure
- `src/pages/`: Content pages (Home, About, Products, Contact)
- `src/components/`: Reusable UI components
- `src/layouts/`: Main site layout (Header, Footer)
- `public/`: Static assets (Images, Fonts)

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```
